# JavaScript Quick Notes

1.   var = function-scoped, let = block-scoped, const = block-scoped + immutable.

2.   Expands arrays/objects into individual elements.

3.  map = transform & return new array, filter = select items, forEach = loop without return.

4. Shorter function syntax, inherits `this` from parent scope.

5. Backtick strings with `${}` for variables & multiline support.
